## 🔖 Stack

<p align="center"><img src="https://user-images.githubusercontent.com/41675375/82345945-d15f1a80-9a30-11ea-92a6-a3b39b3e85f8.png" width="250" height="320"></p>

- Spring Boot: `2.2.6` (Java 8)
- Test: `JUnit4`
- Build: `Gradle`
  ```
  ./gradlew clean build
  ```
- DB: `MariaDB`(AWS RDS)
- Front : `thymeleaf`
- IDE: `IntelliJ IDEA`
- CI/CD: `Travis CI`, `CodeDeploy`(AWS)
